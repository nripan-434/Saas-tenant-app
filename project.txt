This project is an intermediate full-stack AI-powered productivity platform designed to help users organize work through structured workspaces, manage tasks efficiently, and improve productivity using AI-assisted features. The system is built using Next.js for the frontend, Node.js with Express for backend APIs, PostgreSQL for relational data storage, Socket.io for real-time communication, and the OpenAI API for AI-driven functionality. The architecture follows a modular and role-based design to ensure security, scalability, and maintainability.
