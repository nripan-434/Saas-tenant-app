## Project Overview

Ai-Saas-tenant is a multi-tenant project management app with:
- Organization and invitation flow
- Admin/user roles and access guards
- Projects, tasks, and assignments
- AI task generation via Hugging Face Qwen model
- React/Vite frontend with Redux Toolkit
- Express + MongoDB backend
